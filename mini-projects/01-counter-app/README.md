# 01 - Counter App

A simple React counter app built with functional components and the `useState` hook.

## 🚀 Features

- Increment the count
- Decrement the count
- Reset the count to 0

## 🧠 Tech Stack

- React
- JavaScript
- React Hooks (`useState`)

## ▶️ How to Run

```bash
npm install
npm start
```
